This plugin turns the WordPress Gallery into a responsive one using Twitter Bootstrap.

== Description ==

This plugin turns the WordPress Gallery into a responsive one using Twitter Bootstrap. It uses the Magnific Popup to open the images in a responsive lightbox.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wppninja-bootstrap-gallery` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a gallery to your Post / Page / or custom post like you normally would
4. Let the magic begin!

== Frequently Asked Questions ==

Q: Do I need Twitter Bootstrap pre-installed in my theme for this to work?
A: Yes, you will need it to be installed.

Q: Can I change the default image size?
A: Yes, you can, just select the alternative image size from the size drop down menu in the WordPress Gallery manager.

Q: I use a different lightbox, can I disable Magnific Popup?
A: Unfortunately the Magnific Popup is hardcoded into the plugin. Please let me know if it causes any issues.
